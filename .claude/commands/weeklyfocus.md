# Set Weekly Focus

Help me plan and set priorities for the week: $ARGUMENTS

## Instructions

1. **Read current context**:
   - `priorities/weekly-focus.md` — last week's focus (if set)
   - `deals/pipeline.md` — what deals need movement this week
   - `meetings/actions/` — any unresolved action items
   - Calendar for the week using `GOOGLECALENDAR_EVENTS_LIST` for the next 7 days

2. **Interview me** (briefly) if context is missing:
   - What are the biggest things on my mind this week?
   - Any major meetings, deadlines, or decisions?
   - What got deferred last week that still matters?

3. **Suggest a weekly focus** with:
   - Top 3 priorities (specific and achievable this week)
   - Key deals to advance and what "advance" means for each
   - Decisions that need to be made
   - Things to explicitly defer

4. **Show me the draft** and let me adjust

5. **Write to `priorities/weekly-focus.md`** after I confirm — archive the previous week's entry at the bottom

Keep priorities honest and achievable. Three real priorities beat ten aspirational ones.
